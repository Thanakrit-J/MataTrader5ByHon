# 🤖 Trading Bot Monitor

เครื่องมือสำหรับตรวจสอบสถานะบอทเทรดทองคำแบบเรียลไทม์

## 📊 วิธีใช้งาน

### 1. ตรวจสอบสถานะปัจจุบัน
```bash
python simple_status.py
```

### 2. ตรวจสอบอย่างต่อเนื่อง
```bash
python continuous_monitor.py
```
*(กด Ctrl+C เพื่อหยุด)*

### 3. ตรวจสอบจาก Log
```bash
python check_bot_status.py
```

### 4. ดูประวัติการเทรด
```bash
python trade_history.py
```

## 📈 ข้อมูลที่แสดง

- 💰 **ยอดเงินและ Equity**
- 📊 **ตำแหน่งที่เปิดอยู่**
- 📈 **ข้อมูลตลาดทองคำ**
- 🟢🔴 **สัญญาณ BUY/SELL**
- 📋 **การเทรดล่าสุด**
- ⚠️ **ตัวชี้วัดความเสี่ยง**

## 🎯 สีสัญญาณ

- 🟢 **เขียว**: กำไร / สัญญาณ BUY
- 🔴 **แดง**: ขาดทุน / สัญญาณ SELL
- ⚪ **เทา**: รอสัญญาณ / ปกติ

## 🚀 เริ่มใช้งาน

1. เปิด MT5 Terminal และเข้าบัญชี Demo
2. รันบอท: `python trading_bot.py`
3. ตรวจสอบสถานะ: `python simple_status.py`
